```
晚上好：
    
    我现在坐在床上写这文字，被子，枕头都是家里刚寄过来的，冬天很快就要来了，没个厚一点的被子会有点熬不过去的。
    
慧姐，我今天在思考我为什么会写这个东西，到底有啥意义呢。里面全是流水帐，有废话，有抱怨，有犯花痴时的少年思怨，啥都有。

在现实中，其实任何人面对面，这些心理起伏，想法，发生的事情和见过的人，很难再去提起，过去就过去了，会很快就忘记，又或者

有很多想法，特别想去表达，可能时机不合适或者不好意思，就把这想法压下去了。但是我又希望你去了解这人，了解我的生活，

了解我的心路历程，了解我的想法，了解我多么想了解你，这有我想说未说，想说没敢说的话。到时候我把这一放你前面，

这就是我这人的description.多棒呀，有个人属性，也有使用方法，真很不错呢。
  
```
